# 校园旧物公益捐赠回收管理系统 — 技术架构文档

> 技术栈：Spring Boot 3.x · Vue 3 · MyBatis-Plus · MySQL 8.x

---

## 一、总体架构概览

系统采用前后端分离架构，后端提供 RESTful API，前端通过 Axios 调用接口。整体分为三层：

```
┌─────────────────────────────────────┐
│         前端 Vue 3 (SPA)             │  Vite 构建 / Nginx 部署
├─────────────────────────────────────┤
│         后端 Spring Boot 3           │  RESTful API / JWT 认证
├──────────────────┬──────────────────┤
│  MyBatis-Plus    │  Redis 缓存       │  ORM 层 / 会话缓存
├──────────────────┴──────────────────┤
│         MySQL 8.x 数据库             │  主数据存储
└─────────────────────────────────────┘
```

### 4.2 Mapper 接口（继承 BaseMapper）

```java
@Mapper
public interface DonationAppointmentMapper
        extends BaseMapper<DonationAppointment> {

    /**
     * 联表查询：预约列表（含用户名、回收点名称）
     */
    @Select("""
        SELECT a.*, u.real_name as donorName, s.name as stationName
        FROM donation_appointment a
        LEFT JOIN sys_user u ON a.donor_id = u.id
        LEFT JOIN collection_station s ON a.station_id = s.id
        WHERE a.deleted = 0
        AND a.status = #{status}
        ORDER BY a.create_time DESC
    """)
    Page<AppointmentVO> selectAppointmentPage(
            Page<AppointmentVO> page, @Param("status") Integer status);
}
```

### 4.3 Service 层示例

```java
@Service
@RequiredArgsConstructor
public class DonationAppointmentServiceImpl
        extends ServiceImpl<DonationAppointmentMapper, DonationAppointment>
        implements DonationAppointmentService {

    private final CollectionStationService stationService;
    private final SysNotificationService notificationService;
    private final QrCodeUtil qrCodeUtil;

    @Override
    @Transactional(rollbackFor = Exception.class)
    public String createAppointment(AppointmentDTO dto, Long donorId) {
        // 1. 校验回收点容量
        stationService.checkCapacity(dto.getStationId());
        // 2. 生成预约码
        String code = qrCodeUtil.generateCode(donorId, dto.getStationId());
        // 3. 保存预约
        DonationAppointment appointment = BeanUtil.copyProperties(dto, DonationAppointment.class);
        appointment.setDonorId(donorId);
        appointment.setAppointCode(code);
        appointment.setStatus(AppointmentStatus.PENDING);
        save(appointment);
        // 4. 发送预约成功通知
        notificationService.sendNotice(donorId, NoticeType.APPOINT_SUCCESS,
                "您的捐赠预约已提交，预约码：" + code);
        return code;
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void volunteerConfirm(String appointCode, Long volunteerId) {
        DonationAppointment appoint = lambdaQuery()
                .eq(DonationAppointment::getAppointCode, appointCode)
                .one();
        if (appoint == null || appoint.getStatus() != AppointmentStatus.PENDING) {
            throw new BusinessException("预约码无效或已处理");
        }
        lambdaUpdate()
                .eq(DonationAppointment::getId, appoint.getId())
                .set(DonationAppointment::getStatus, AppointmentStatus.RECEIVED)
                .set(DonationAppointment::getVolunteerId, volunteerId)
                .update();
        // 通知捐赠者
        notificationService.sendNotice(appoint.getDonorId(),
                NoticeType.ITEM_RECEIVED, "您的旧物已被志愿者接收，正在整理中");
    }
}
```

### 5.3 Axios 封装（request.ts）

```typescript
import axios from 'axios'
import { useUserStore } from '@/stores/user'
import router from '@/router'
import { ElMessage } from 'element-plus'

const request = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || '/api',
  timeout: 10000,
})

// 请求拦截：自动附加 JWT
request.interceptors.request.use((config) => {
  const userStore = useUserStore()
  if (userStore.token) {
    config.headers.Authorization = `Bearer ${userStore.token}`
  }
  return config
})

// 响应拦截：统一处理错误
request.interceptors.response.use(
  (response) => {
    const res = response.data
    if (res.code !== 200) {
      ElMessage.error(res.message || '请求失败')
      return Promise.reject(new Error(res.message))
    }
    return res.data
  },
  (error) => {
    if (error.response?.status === 401) {
      useUserStore().logout()
      router.push('/login')
    }
    ElMessage.error('网络异常，请稍后重试')
    return Promise.reject(error)
  }
)

export default request
```

## 七、安全方案

### 7.1 认证流程

```
用户登录 → Spring Security 验证账号密码
         → 生成 JWT（含 userId, userType, exp）
         → 前端存入 localStorage / Pinia
         → 每次请求 Header 携带 Authorization: Bearer <token>
         → 后端 JwtFilter 解析验证 → 注入 SecurityContext
```

### 7.2 角色权限控制

使用 `@PreAuthorize` 注解进行方法级别权限控制：

```java
// 仅管理员可访问
@PreAuthorize("hasRole('ADMIN')")
@DeleteMapping("/user/{id}")
public R<Void> deleteUser(@PathVariable Long id) { ... }

// 志愿者或管理员可访问
@PreAuthorize("hasAnyRole('VOLUNTEER', 'ADMIN')")
@PutMapping("/item/{id}/status")
public R<Void> updateItemStatus(...) { ... }
```

## 八、预约码与电子证明生成

### 8.1 预约码（二维码）

使用 ZXing 将预约码字符串编码为二维码图片，前端展示；志愿者扫描后向后端提交该 Code 进行核验。

```java
// 生成规则：DONATE-{userId}-{timestamp}-{随机4位}
public String generateCode(Long userId, Long stationId) {
    return String.format("DONATE-%d-%d-%04d",
            userId, System.currentTimeMillis(), RandomUtil.randomInt(1000, 9999));
}
```

### 8.2 电子证明（PDF）

使用 iText 7 动态生成 PDF，包含捐赠者姓名、捐赠物品、时间、公益用途、系统盖章，支持下载，可用于志愿时长认定。

```java
public byte[] generateCertificate(CertificateDTO dto) {
    try (ByteArrayOutputStream baos = new ByteArrayOutputStream()) {
        PdfWriter writer = new PdfWriter(baos);
        PdfDocument pdfDoc = new PdfDocument(writer);
        Document document = new Document(pdfDoc);

        // 加载中文字体（防止乱码）
        PdfFont font = PdfFontFactory.createFont("STSong-Light", "UniGB-UCS2-H");

        document.add(new Paragraph("校园公益捐赠证明").setFont(font).setFontSize(20));
        document.add(new Paragraph("捐赠者：" + dto.getDonorName()).setFont(font));
        document.add(new Paragraph("捐赠物品：" + dto.getItemDesc()).setFont(font));
        document.add(new Paragraph("公益用途：" + dto.getPurpose()).setFont(font));
        document.add(new Paragraph("证书编号：" + dto.getCertNo()).setFont(font));
        document.close();
        return baos.toByteArray();
    }
}
```

**本地开发启动顺序：**

1. 启动 MySQL，执行建表 SQL 初始化数据库
2. 启动 Redis
3. 运行 Spring Boot：`mvn spring-boot:run`
4. 启动 Vue 前端：`npm run dev`（默认 5173 端口，代理 /api → 8080）

**vite.config.ts 代理配置：**
